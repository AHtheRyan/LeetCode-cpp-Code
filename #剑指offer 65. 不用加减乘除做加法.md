### 不用加减乘除做加法

> 写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。  

----------

#### I 位运算

基于加法器可以知道，位运算的两数相加可以转为**与运算**与**异或运算**的集合，其中  
- 与运算的结果为当前位
- 异或运算的结果为进位  

因此，可以通过此种方式完成位运算  

```cpp
int add(int a, int b) 
{
    while(b != 0)
    {
        int flag = (unsigned int)(a & b) << 1;
        a ^= b;
        b = flag;
    }
    return a;
}
```
